# Projeto -- Base de Conhecimento CEFOR/IFes

Fonte canonica para identidade, equipe, premissas e plataforma.

## Visao

Criar uma nova Base de Conhecimento muito bem estruturada, bonita, com experiencia de leitura excelente, extremamente util e funcional para profissionais de EaD do IFes.

## Proposito-Farol

> "A Base de Conhecimento do CEFOR e o lugar onde qualquer profissional de EaD do IFES resolve problemas com autonomia e se forma continuamente."

Duas funcoes em uma so base:
- **Consulta:** "Estou com um problema agora, preciso da resposta"
- **Formacao:** "Quero aprender, quero evoluir"

## Premissas

- WordPress e a plataforma. Nao havera avaliacao de alternativas.
- O servidor novo ja esta pronto com WordPress atualizado e instalado.
- O conteudo sera migrado no final, apos as definicoes estarem prontas.
- Os ~60 artigos atuais provavelmente serao reescritos seguindo o novo padrao.
- O foco e qualidade e estrutura, nao velocidade de entrega.

## Publico-Alvo

Profissionais que atuam na educacao a distancia ou uso de tecnologias educacionais no IFES: mediadores, professores, coordenadores, tecnicos.

- **CMS:** WordPress (servidor novo, ja instalado)
- **Base atual:** https://conhecimento.cefor.ifes.edu.br/
