# Base de Conhecimento CEFOR/IFes

Workspace para o redesign da Base de Conhecimento do CEFOR. Cada stage corresponde a uma fase do projeto, e o output de cada stage alimenta o próximo.

## Estrutura

```
Base-conhecimento-oficial/
├── _config/               Identidade do projeto, pilares e decisões
├── setup/                 Onboarding do projeto
├── stages/
│   ├── 00-benchmarking/   Fase 0 — Pesquisa e benchmarking [CONCLUÍDA]
│   ├── 01-fundacoes/      Fase 1 — Content System + Taxonomia + Descoberta
│   ├── 02-design-system/  Fase 2 — Visual, leitura, acessibilidade, interação
│   ├── 03-implementacao/  Fase 3 — Tema WordPress + componentes + templates
│   ├── 04-conteudo/       Fase 4 — Produção e reescrita de artigos
│   └── 05-lancamento/     Fase 5 — Lançamento e operação contínua
├── shared/                Recursos cross-stage: roadmap, ata, transcrição
└── data/                  Dados brutos: inventário base antiga, URLs
```
